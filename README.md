## 📌 Proposta de Valor

O **ReCiclo** é um aplicativo criado para conectar pessoas que desejam **doar objetos** com pessoas ou entidades que **necessitam desses materiais**, promovendo a sustentabilidade e a reutilização de itens.

O principal benefício para o usuário é a facilidade de:

- Encontrar itens próximos da sua localização;
- Doar objetos de forma simples e rápida;
- Conversar diretamente com interessados;
- Contribuir com o meio ambiente;
- Receber recompensas através da gamificação com **Selos Eco**.

---

## 📱 Descrição das Telas

### 🏠 Tela Home / Feed
A tela inicial do aplicativo exibe diversos itens disponíveis para doação em formato de lista/grid.

Nela o usuário pode:

- Visualizar imagem do item;
- Ver categoria;
- Ver localização;
- Selecionar um item para mais detalhes.

---

### 📦 Tela de Detalhes do Item
Ao clicar em um item no feed, o usuário acessa os detalhes.

Nessa tela contém:

- Foto do item;
- Categoria;
- Descrição;
- Informações do doador;
- Botão **“Tenho interesse”** para iniciar contato.

---

### 💬 Tela de Chat / Mensagens
Após clicar em “Tenho interesse”, o usuário é direcionado para a conversa.

Funções da tela:

- Enviar mensagens;
- Tirar dúvidas;
- Combinar local e horário para retirada.

---

### 👤 Tela de Perfil
Tela destinada às informações do usuário.

Nela o usuário pode visualizar:

- Foto de perfil;
- Informações pessoais;
- Possíveis conquistas/selos ecológicos.

---

### ➕ Tela de Cadastro de Item
Tela utilizada para cadastrar um novo item para doação.

O usuário pode:

- Adicionar fotos;
- Inserir título;
- Descrição;
- Categoria do item.

---

## 🚶 Jornada Principal do Usuário

A jornada principal no aplicativo funciona da seguinte forma:

1. O usuário abre o aplicativo e acessa a **Tela Home**.
2. Navega pelos itens disponíveis próximos da sua localização.
3. Seleciona um item do interesse.
4. É direcionado para a **Tela de Detalhes do Item**.
5. Clica em **“Tenho interesse”**.
6. Vai para a **Tela de Chat** para conversar com o doador.
7. Combina retirada ou entrega do item.
8. Após a conclusão, pode acessar seu **Perfil** e acompanhar conquistas.

---

## 🎯 Objetivo do Projeto

O objetivo do ReCiclo é incentivar a **economia circular**, diminuir o desperdício e facilitar a conexão entre doadores e interessados de forma simples e intuitiva.
